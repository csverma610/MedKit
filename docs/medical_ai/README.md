## How to Use These Documents

Each module document contains:
- **Overview:** Quick summary of what the module does
- **Scope:** Exactly what the module covers
- **What It Does:** Specific capabilities and features
- **Why It Matters:** Real-world benefits to the user
- **How to Use:** Code examples and command-line usage
- **Important Notes:** Critical information and limitations
- **Limitations:** Honest assessment of what the module can't do
- **Disclaimer:** Legal and safety information

---

## Key Principles

**Objective:** Facts-based, not marketing-focused descriptions.

**Clear:** Written in plain language without unnecessary jargon.

**Practical:** Focus on real-world applications and benefits.

**Safe:** Emphasize limitations and when to see professionals.

**User-Focused:** Assume reader has limited time and memory.

---

## Important Reminders

**Educational Use Only:** These documents provide information for learning.

**Not Medical Advice:** Not a substitute for professional medical consultation.

**See a Doctor:** For any health concern, consult qualified healthcare providers.

**Individual Variation:** Medical information applies differently to different people.
