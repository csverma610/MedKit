## Common Questions

**How long do test results take?**
Usually 24 hours to several days; some available immediately. Ask your doctor.

**Why do I need multiple tests?**
Doctors often use test sequences to confirm diagnosis and rule out other conditions.

**Will tests hurt?**
Most diagnostic tests cause minimal discomfort. Your doctor will explain.

**Can I eat/drink before my test?**
Depends on the test. Ask beforehand about preparation.

**What if my test result is abnormal?**
Contact your doctor. One abnormal result usually requires confirmation before diagnosis.

**Do I need to worry about radiation?**
Radiation from medical tests is safe at diagnostic doses. Benefits usually outweigh risks.
